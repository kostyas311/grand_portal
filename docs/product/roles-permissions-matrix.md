# Матрица ролей и прав доступа

## Роли (упрощённые)

| Роль | Код | Описание |
|------|-----|---------|
| Пользователь | `USER` | Стандартный пользователь системы |
| Администратор | `ADMIN` | Расширенные права управления системой |

## Что могут все пользователи (USER и ADMIN)

- Авторизоваться и работать в системе
- Просматривать справочник источников данных
- Создавать карточки
- Просматривать карточки (свои + назначенные)
- Редактировать незакрытые карточки
- Назначать исполнителей и проверяющих на карточке
- Менять статусы карточек (NEW→IN_PROGRESS→REVIEW→DONE, возврат из REVIEW)
- Прикреплять исходные материалы (файлы и ссылки)
- Загружать результаты (файлы и ссылки)
- Скачивать файлы (исходные и результаты)
- Оставлять комментарии проверки
- Просматривать историю действий
- Просматривать главную страницу с готовыми проектами
- Редактировать свой профиль (ФИО, пароль)

## Что может только ADMIN

- Просматривать список всех пользователей системы
- Создавать новых пользователей
- Редактировать данные любого пользователя
- Блокировать / разблокировать пользователей (`is_active`)
- **Физически удалять** карточки (hard delete)
- **Физически удалять** файлы и записи
- Редактировать справочник источников (создание, изменение, архивирование)
- Просматривать системный журнал аудита
- Сбрасывать пароль любого пользователя
- Отменять карточки в любом статусе

## Матрица прав

### Пользователи

| Действие | USER | ADMIN |
|----------|------|-------|
| Список всех пользователей | ❌ | ✅ |
| Создание пользователя | ❌ | ✅ |
| Редактирование своего профиля | ✅ | ✅ |
| Редактирование чужого профиля | ❌ | ✅ |
| Блокировка пользователя | ❌ | ✅ |
| Сброс пароля пользователя | ❌ | ✅ |

### Справочник источников

| Действие | USER | ADMIN |
|----------|------|-------|
| Просмотр | ✅ | ✅ |
| Создание источника | ❌ | ✅ |
| Редактирование источника | ❌ | ✅ |
| Архивирование/восстановление | ❌ | ✅ |

### Карточки

| Действие | USER | ADMIN |
|----------|------|-------|
| Просмотр всех карточек | ✅* | ✅ |
| Создание карточки | ✅ | ✅ |
| Редактирование карточки (открытой) | ✅ | ✅ |
| Редактирование закрытой карточки | ❌ | ❌** |
| Отмена карточки | ✅ | ✅ |
| **Физическое удаление карточки** | ❌ | ✅ |

*USER видит все карточки в списке, но может редактировать только те, где участвует
**Закрытые карточки (DONE/CANCELLED) не редактируются никем

### Статусы

| Переход | USER | ADMIN |
|---------|------|-------|
| NEW → IN_PROGRESS | ✅ | ✅ |
| IN_PROGRESS → REVIEW | ✅ (нужен результат) | ✅ |
| REVIEW → DONE | ✅ | ✅ |
| REVIEW → IN_PROGRESS | ✅ (нужен комментарий) | ✅ |
| ANY → CANCELLED | ✅ | ✅ |

### Файлы

| Действие | USER | ADMIN |
|----------|------|-------|
| Прикрепить файл/ссылку | ✅ | ✅ |
| Скачать файл | ✅ | ✅ |
| Удалить файл (из открытой карточки) | ✅* | ✅ |
| **Физическое удаление файла** | ❌ | ✅ |

*только если карточка не закрыта
