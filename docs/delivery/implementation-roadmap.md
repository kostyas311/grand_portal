# Roadmap реализации — Нормбаза

## Статус

Проект находится в рабочем состоянии: основная бизнес-функциональность реализована, портал запускается через Docker и используется как внутренняя система сопровождения карточек, инструкций и обращений.

## Что уже реализовано

### Платформа и эксплуатация

- `docker-compose.yml` для `postgres`, `api`, `web`, `nginx`
- единый `Dockerfile` монорепо
- автоматическая синхронизация схемы БД и seed на старте API
- базовая обработка ошибок API
- rate limiting через `@nestjs/throttler`

### Карточки и процессы

- создание, редактирование и просмотр карточек;
- статусы `NEW`, `IN_PROGRESS`, `REVIEW`, `DONE`, `CANCELLED`;
- исполнитель, проверяющий, подписчики;
- список карточек и канбан-доска;
- история изменений;
- исходные материалы;
- версии результатов;
- комментарии и возврат с замечаниями;
- отчёты по карточкам;
- спринты и привязка карточек к спринтам.

### Внутренние сервисы

- центр уведомлений;
- email-настройки уведомлений;
- обращения к администратору;
- база инструкций с папками и вложениями;
- справочник источников;
- профили пользователей и переключение темы.

## Команды для быстрого старта

```bash
cp .env.example .env
mkdir -p storage
docker compose up -d --build
```

## Команды для локальной разработки

```bash
cd apps/api
npm install
npx prisma generate
npm run start:dev
```

```bash
cd apps/web
npm install
npm run dev
```
